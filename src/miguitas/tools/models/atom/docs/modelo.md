# Modelo Atómico visto como nodo

Para entender al átomo de manera abstracta, puede ser como un nodo, sus entradas son los espacios para electrones disponibles.

Estructura del nodo:

Hidrogeno
configuración electrónica: 1s1

con una entrada disponible.

Carbono
configuración electrónica: 1s2 2s2 2p2

tiene 4 entradas disponibles.
